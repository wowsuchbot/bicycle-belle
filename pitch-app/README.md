# Bicycle Belle Pitch App

A Next.js application for pitching Bicycle Belle - the AI-powered local bike shop discovery platform.

## Features

- Interactive pitch slideshow presentation
- Demo prototype with mock product data
- Lightspeed-style API integration showcase

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `/` - Pitch slideshow presentation
- `/demo` - Interactive demo with mock products

## Built With

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
